Project Details

The Problem

Unexpected rain can ruin clothes left outside to dry, causing inconvenience, potential damage, and the need to rewash items. Traditional methods rely on constant human monitoring and quick reaction times.

Our Solution

An automated system that detects rain using sensors and immediately activates a servo motor to pull clothes to safety. Complete with audio announcements and visual LED alerts for comprehensive notification.

How It Works

Rain Detection

Rain sensor continuously monitors weather conditions and triggers the system when moisture is detected.

Servo Activation

Servo motor immediately engages to pull the clothesline mechanism, bringing clothes to safety.

Alert System

DFPlayer Mini plays audio announcement while LEDs provide visual indication of system status.

LCD Display

Real-time status updates shown on I2C LCD display for system monitoring.
